### Sobre o ROCKET.Q
Projeto desenvolvido para gerenciar perguntas anonimas de usuários direcionadas ao dono da página. Projeto desenvolvido na semana NLW Rocketseat na missão discover.

## getting started
```
- Clone este repositório
- npm install
- npm run init-db
- npm run start
```
